Why Nova

Built for the modern era

Nova takes the best ideas from the last 30 years of language design, and throws away everything else.

⚡

Blazing Fast

Nova compiles to native machine code with zero garbage collection pauses. Comparable to C++ performance with a fraction of the complexity.

🛡️

Memory Safe

A strict ownership model eliminates null pointer errors, data races, and use-after-free bugs at compile time — before your code even runs.

✨

Expressive Syntax

Write code that reads like plain English. Nova's clean syntax reduces boilerplate and lets you focus on solving problems, not fighting the language.

🔄

First-class Async

Async/await built into the language core. Write concurrent code that looks synchronous. No callback hell, no promise chains.

📦

Rich Standard Library

HTTP, JSON, crypto, file I/O, and more — all batteries included. Stop hunting for packages for basic tasks.

🧩

Interoperable

Call C libraries directly. Compile to WebAssembly. Generate bindings for Python and Node.js. Nova plays well with your existing stack.

Nova vs. the alternatives

See how Nova's clean syntax compares to writing the same code in other popular languages.

Python
def fetch_users(active=True):
  users = db.query("SELECT * FROM users")
  if active:
    users = [u for u in users if u["active"]]
  return users
 
# No types, no compile-time safety
result = fetch_users()
print(result[0]["nmae"]) # typo! crashes at runtime
Nova ✓ Safer
fn fetchUsers(active: Bool = true) -> []User {
  let users = db.query<User>("SELECT * FROM users")
  return active ? users.filter(\u => u.active) : users
}
 
// Types checked at compile time
let result = fetchUsers()
print(result[0].nmae) // caught before running!

Quick Start

Up and running in minutes

From installation to your first running program — it takes less than 5 minutes.

1

Install Nova

One command installs the Nova compiler, package manager, and LSP server for your editor.

$ curl -fsSL novalang.dev/install | sh
2

Create a project

The nv new command scaffolds a project with sensible defaults, ready to run.

$ nv new my-app && cd my-app
3

Run your code

Build and run in one step. Nova compiles fast enough that it feels like an interpreted language.

$ nv run main.nv
